## 🚀 Featured Projects

### ⚙️ 2-Bit ALU — Digital Logic Design
The Arithmetic Logic Unit is the part of every CPU that actually does the "computing" — addition, subtraction, logic operations, all of it runs through here. Modern processors use 32-bit or 64-bit ALUs, but I built mine at **2-bit scale** on purpose: small enough to design, wire, and fully understand every gate of the circuit by hand, while still capturing exactly how a real ALU works. It was less about raw computing power and more about proving I understood computing at its most fundamental level, gate by gate.

### ❄️ Smart AC Controller — PIC16F877A · Assembly (MPASM)
A fully custom embedded system built from scratch — both the circuit and the firmware. Reads temperature via an LM35 sensor, drives an HD44780 LCD, controls a PWM fan and SG90 servo, and supports four operating modes (HEAT, COLD, AUTO, SLEEP). Every line of logic is written in raw Assembly — no libraries, no shortcuts.

### 📊 Data Analytics System — Python · Tkinter
A desktop application for data analysis with a custom-built GUI in Tkinter, using NumPy and Matplotlib under the hood for processing and visualization. Built to turn raw data into readable insight without relying on a heavyweight framework.

### 🛒 Minimarket POS System — C++ · Qt · CSV
A complete point-of-sale desktop application with barcode scanner support, a custom linked-list data layer, CSV-based persistence, and a category-based OOP architecture — six item types derived from a shared base class — plus a financial reporting module.

### 📚 Library Management System — C · BST · Queue · Linked List
A data structures lab project implementing a working library system using binary search trees, queues, and linked lists.
